# 🚀 AI Startup Success Predictor

An AI-powered system that predicts startup success, risk levels, and funding potential using Machine Learning.

---

## 📊 Features
- 🔮 Success Prediction (Success / Failure)
- 📉 Risk Scoring (Low / Moderate / High)
- 💰 Funding Estimation
- 📂 Upload CSV/XLSX datasets
- ⚡ Real-time analysis dashboard

---

## 🧠 Tech Stack
- Python (FastAPI)
- Scikit-learn (Random Forest)
- Pandas / NumPy
- React + Vite (Frontend)

---

## ⚙️ How to Run

### Backend
```bash
pip install -r requirements.txt
uvicorn main:app --reload

Frontend

cd frontend
